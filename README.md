# Spaceship Titanic - Kaggle Competition

Repositório dedicado aos estudos e experimentos da competição Spaceship Titanic no Kaggle.

O foco inicial do projeto é realizar uma análise exploratória dos dados (EDA) e estruturar um pipeline organizado para futuras etapas de modelagem.

## Estrutura do Projeto

```
.
├── data/
├── notebooks/
├── src/
├── submissions/
├── .gitignore
├── requirements.txt
└── README.md
```

# Ambiente e Instalação

O projeto utiliza ambiente virtual (`venv`) e gerenciamento de dependências via `requirements.txt`.

# Criando o ambiente

### Linux/macOS

```bash
python -m venv venv
source venv/bin/activate
```

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

## Instalando dependências

```bash
pip install -r requirements.txt
```
